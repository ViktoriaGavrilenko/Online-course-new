# Online Course Platform

A responsive web application for an online course platform built with modern frontend technologies.

## 🔹 Project Description
This project is a frontend application that demonstrates a responsive UI for an online learning platform. It focuses on clean layout, reusable components, and adaptive design across different screen sizes.

## 🚀 Features
- Responsive design (mobile, tablet, desktop)
- Component-based architecture
- Modern UI built with Tailwind CSS
- Fast development and build with Vite
- Type-safe code using TypeScript

## 🛠️ Tech Stack
- React.js
- TypeScript
- Vite
- Tailwind CSS

## 📁 Project Structure
- Component-based architecture
- Tailwind utility classes for styling
- Semantic HTML5 markup
